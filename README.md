# Inventory Management System (C++)

## Overview
This project implements a **menu-driven Inventory Management System** using C++.  
The system allows users to manage products, update stock quantities, and view inventory reports.

It demonstrates the use of **arrays, structures, loops, and conditional statements** to simulate a real-world inventory management scenario.

## Features
- Add new products
- Update stock quantities
- Display inventory details
- Calculate total inventory value
- Menu-driven console interface

## Technologies Used
- C++
- Standard C++ Libraries
- Console-based application

## System Components

### Product Structure
Each product stores:
- Product ID
- Product Name
- Quantity
- Price

### Inventory Operations
- Add Product
- Update Stock
- View Inventory
- Exit Program

## Example Program Flow
1. Start program
2. Display menu options
3. User selects operation
4. Perform action:
   - Add product
   - Update stock
   - View inventory
5. Repeat until user exits
